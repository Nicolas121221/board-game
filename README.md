# 🕹️ Pixel art in canvas

This is a personal project I developed to **improve my skills in vanilla JavaScript** and gain a deeper understanding of the **HTML5 Canvas API**.
The goal of this project is to build several **classic board games** — such as chess and checkers — using only JavaScript, Canvas, and custom **pixel art** visuals.

All rendering, input handling, and game logic are implemented manually, without any external frameworks.
This approach allows full control over every layer of the system, from how sprites are drawn to how pieces move across the board.
